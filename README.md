# Me personal blog

|                |                         |
| -------------- |:-----------------------:|
| URL:           | http://carlosbecker.com |
| Deployable on: | Github Pages            |
| Dependencies:  | Bundler, npm, bower     |



## Setup

```bash
bundle
npm install grunt-cli bower
npm install
bower install
```

## Watch

```bash
grunt watch
rake preview
```

## Update assets

```bash
bower update
grunt
```
